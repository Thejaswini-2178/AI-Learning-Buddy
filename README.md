# 🎓 Hook Mate – AI Learning Buddy

Hook Mate is an AI-powered Learning Buddy built using Streamlit and Google Gemini AI. It helps beginners learn React Hooks through simple explanations, real-life examples, interactive quizzes, and instant feedback.

## 🚀 Features

- 📖 Explain React Hooks in simple language
- 🌍 Real-life examples for better understanding
- 📝 Interactive 5-question quizzes
- ✅ Instant quiz evaluation with explanations
- 📊 Learning Progress Dashboard
- 📥 Download AI notes as TXT or PDF
- 🎯 Beginner, Intermediate, and Advanced difficulty levels

## 🛠️ Technologies Used

- Python
- Streamlit
- Google Gemini API
- JSON Repair
- FPDF2
- Python Dotenv

## ▶️ Run Locally

1. Clone the repository:

```bash
git clone https://github.com/Thejaswini-2178/AI-Learning-Buddy
```

2. Navigate to the project folder:

```bash
cd Hook-Mate-AI-Learning-Buddy
```

3. Install the required packages:

```bash
pip install -r requirements.txt
```

4. Create a `.env` file and add your Gemini API key:

```env
GEMINI_API_KEY=YOUR_API_KEY
```

5. Run the application:

```bash
streamlit run app.py
```
