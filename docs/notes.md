4. Sample Learning Conversation 
Learner: Hi HookMate! Can you teach me about React Hooks? 
HookMate: Of course! 😊 
React Hooks are special functions introduced in React that allow developers to use state and 
other React features inside functional components without writing class components. 
Some commonly used React Hooks are: 
 useState – Used to store and update data. 
 useEffect – Used to perform side effects like fetching data or updating the page. 
 useContext – Used to access shared data across components. 
 useRef – Used to directly access DOM elements or store mutable values. 
React Hooks make your code simpler, more readable, and easier to maintain. 
Learner: Can you give me a real-life example? 
HookMate: Sure! 
Imagine you're using a music player on your phone. 
 The current song being played is like the state of your application. 
 When you tap the Next button, the current song changes. 
 In React, the useState Hook helps manage and update this changing information. 
Just as your music player remembers which song is currently playing, useState remembers 
the current value in your React component.

Learner: My answer is B.

React Hooks allow developers to use features such as state and lifecycle functionality inside

5. Five-Question Quiz + Answers 
Topic: React Hooks 
Question 1: What is the primary purpose of React Hooks? 
A. To design web pages 
B. To use state and other React features in functional components 
C. To connect to databases 
D. To create CSS animations 
Correct Answer: B 
Explanation: React Hooks allow developers to use state and other React features inside 
functional components without using class components. 
Question 2: Which React Hook is commonly used to manage state? 
A. useEffect 
B. useState 
C. useRef 
D. useContext 
Correct Answer: B 
Explanation: The useState Hook is used to create and update state variables in a functional 
component. 
Question 3: Which Hook is mainly used to perform side effects such as fetching data? 
A. useRef 
B. useContext 
C. useEffect 
D. useMemo 
Correct Answer: C 
Explanation: The useEffect Hook is used for side effects like API calls, timers, and updating 
the document title. 
Question 4: Which Hook allows components to access shared data without passing props 
through multiple levels? 
A. useContext 
B. useState 
C. useReducer 
D. useMemo 
Correct Answer: A 
Explanation: The useContext Hook provides access to shared data from a React Context. 
Question 5: Which statement best describes React Hooks? 
A. They can only be used in class components. 
B. They replace JavaScript. 
C. They allow functional components to use React features like state and lifecycle 
functionality. 
D. They are only used for styling. 
Correct Answer: C 
Explanation: React Hooks make functional components more powerful by enabling them to 
use features that were previously available only in class components. 
6. Reflection on AI Limitations 
Developing the AI Learning Buddy helped me understand how artificial intelligence can make 
learning more interactive and engaging. In this project, I chose React Hooks as the learning 
topic because it is an important concept for beginners learning React. The AI Learning Buddy 
was able to explain React Hooks in simple language, provide a practical real-life example, 
generate quiz questions, and offer feedback on the learner's answers. These features make 
learning easier and encourage users to understand concepts step by step. 
One of the biggest strengths of AI as a tutor is its ability to provide instant responses and 
personalized explanations. It is available at any time and can simplify difficult concepts for 
beginners. The quiz and feedback features also help learners evaluate their understanding 
and identify areas that need improvement. During the sample conversation, the AI explained 
React Hooks clearly and used a relatable example, making the concept easier to understand. 
However, AI also has some limitations. Sometimes the explanations may be too general or 
may not match the learner's exact level of knowledge. AI can occasionally provide inaccurate 
or outdated information, especially if the topic changes over time. It also cannot fully 
replace a human teacher, who can understand emotions, learning styles, and provide real
world guidance based on personal experience. 
To improve AI learning tools, they should include more interactive features such as progress 
tracking, personalized learning paths, visual diagrams, and adaptive quizzes based on the 
learner's performance. Regular updates to the AI model and reliable sources of information 
can also improve the accuracy of responses. Overall, this project showed that AI can be an 
effective learning assistant when used responsibly alongside traditional learning methods.
